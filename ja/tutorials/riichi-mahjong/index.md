# 序

<ruby>立直<rt>リーチ</rt>麻雀児<rt>マージャン</rt></ruby>[^1]チュートリアルシリーズへようこそ。リーチ麻雀は今日、ほぼ「日本麻雀」と同義として扱われており、その名称は代表的なルールである「リーチ」に由来します。

[^1]: 中国語において「マージャン」は「麻将」と表記されることが多く、発音はほぼ同一である。一般に「ジャン」は、北部呉語における「<ruby>雀<rt>チャッ</rt></ruby>」の<ruby>児<rt>ン</rt></ruby>化に由来すると考えられている。参照：https://en.wikipedia.org/wiki/Erhua#Wu

本シリーズは、私自身の麻雀学習に合わせて更新予定です。まず基本ルールを扱い、その後に戦術へと進みます。麻雀未経験の読者にも適した内容ですが、細部の例外規則や「なぜそうなるのか」に関する解説も含んでおり、初心者は必要に応じて読み飛ばしていただければと思います。
