# 序

<ruby>立<rt>リー</rt>直<rt>チ</rt>麻<rt>マー</rt>雀<sub>兒</sub><rt>ジャン</rt></ruby>[^1]チュートリアルへようこそ。現在では「立直麻雀」（<l-ja>リーチ<ruby>麻雀<rt>マージャン</rt></ruby></l-ja>）は、ほぼ「日本麻雀」と同義に使われます。この呼び名は、代表的なルールである立直に由来します。

[^1]: マージャンは中国語でよく「麻將」と書き、発音は同じです。一般に、「ジャン」は北部呉語における「<ruby>雀<rt>ジャク</rt></ruby>」の兒化した形を表していると考えられている。参照：https://en.wikipedia.org/wiki/Erhua#Wu

このシリーズは、私自身の麻雀学習と並行して更新していく予定です。まず基本ルールを扱い、その後に戦術を解説します。麻雀未経験の読者でも読めるようにしつつ、細かな例外規則や、なぜそうなるのかという話題も含めます。初心者の方は必要に応じて読み飛ばしてください。
