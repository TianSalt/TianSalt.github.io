# 1.3 リーチ

いよいよ「タイトル回収」の節にやってきました。この「リーチ麻雀」において、リーチは単なる特徴的なシステムであるだけでなく、他の麻雀と比べても核心的なメカニズムと言えます。前述の通り、リーチは状態役であり、メンゼンでテンパイしてさえいれば使用できます。これにより、本来は手役のない手牌でもメンゼンのまま手を進める道が開かれるだけでなく、すでに手役がある場合でもさらに翻数を重ねることができます。また、日本麻雀には多くの「**メンゼン役**」が存在し、手牌に副露があると成立しません。そのため、これらの役はリーチとセットで現れることが多くなります。本節ではリーチの仕組みとさらに多くの役を紹介していきますが、その前に甜Saltから、まずは点数に関するお話をさせてください。何と言っても、日本麻雀は最終的な順位を競うゲームであり、その順位は持ち点によって決まるからです。手牌の得点の高さ――戦術的にはよく「<ruby>打<rt>だ</rt>点<rt>てん</rt></ruby>」と呼ばれます――について具体的なイメージを持ってもらうために、まずは満貫以上の得点計算から説明します。この部分の計算は最もシンプルで、初心者の方が最も早く馴染む部分でもあります。なお、本節では一般的なルールのみに焦点を当て、競技ルールとの違いについては次節で説明します。

## 1.3.1 満貫

実戦においては、1.2.1で述べた「基本点」を気にすることなく、点数早見表だけを見るようになります。しかし原理から言うと、基本点が2,000点に達した状態を「満貫」と呼びます。古典麻雀における満貫は最高得点の打ち止めでしたが、その後の「インフレ」を経て、現代の仕組みでは、基本点が2,000点を超えると符数を考慮せず翻数だけで計算するようになっています。5翻までは基本点が2,000点で頭打ちとなり、6〜7翻の手牌は3,000点、8〜10翻は4,000点、11〜12翻は6,000点というように固定されます。次節で分かりますが、満貫未満の「翻」は文字通り点数が2倍になっていくのに対し、満貫以上になるとその上昇は急激に緩やかになります。具体的に、満貫以上の点数早見表は以下の通りです。

> **凡例**：**満貫**の欄を例にすると、**8,000**：子がロンアガリしたときに放銃者が支払う点数；**2,000-4,000**：子がツモアガリしたときに子、親が支払う点数；**12,000**：親がロンアガリしたときに放銃者が支払う点数；**4,000**：親がツモアガリしたときに子が支払う点数。

**役満**（競技ルールの文書などでは「四倍満」とも呼ばれます）は、現代の日本麻雀における得点の上限と言えます。大三元、四暗刻、国士無双などの希少な役を完成させた場合は直接役満となり、このときは役満以外の役はすべて無視され、一律で基本点8,000点として計算されます（詳細は次節で詳述します）。一方、通常の役の積み重ねによって13翻以上に達した場合は「**数え役満**」と呼ばれます。

早見表における得点の表示方法は、アガった人が点数を申告する際の言い方でもあります。例えば、親の満貫[^oyaman]のロンアガリなら「12,000」、親の満貫のツモアガリなら「4,000オール」、子の跳満のツモアガリなら「3,000-6,000」などと発声します。積み場の場合は、後ろに「は」を繋げて、本場を含めた点数を申告します。例えば1本場であれば「12,000は12,300」「4,000オールは4,100オール」「3,000-6,000は3,100-6,100」のようになります。

## 1.3.2 リーチ

<p><ruby>立直<rt>リーチ</rt></ruby>（語源とは異なりますがreachとも表記されます）は、日本麻雀において最も特徴的なメカニズムです。メンゼンでテンパイしていればリーチする（リーチをかける）ことができます。具体的には、任意のツモ番の後、以下の条件をすべて満たしている場合に可能です。</p>

1. メンゼンであること（チー、ポン、明槓をしていない）。
2. テンパイしていること。
3. 持ち点が1,000点以上あること<small>（トビありルールの場合）</small>。
4. 牌山の残りが4枚以上あること<small>（競技ルールではこの制限はなく、いわゆる「ツモ番のないリーチ」も認められますが、ただ海底牌をツモった後はリーチできないというだけです）</small>。

手順は以下の通りです。

::: theorem
1. 「**リーチ**」と発声する。
2. テンパイとなる牌を、河に横向きに打牌する。これを「リーチ宣言牌」と呼ぶ<small>（この宣言牌が鳴かれた場合は、次の巡目の打牌を横向きにします。ロンされた場合は、第3ステップの手順は行いません）</small>。
3. 卓の中央に千点棒を1本供託し、これを「リーチ棒」と呼ぶ。
4. その局が終了するまで、打牌を選ぶことはできず、ツモった牌をそのまま打つこと（ツモ<ruby>切<rt>ぎ</rt></ruby>り[^tumogiri]）しかできない。
:::

[^tumogiri]: 対義語として、ツモ牌以外の元々手牌にあった牌を打つことを「手出し」と呼びます。この両者を区別するため、日本麻雀では一般的に、まず打牌を完了してから、ツモった牌を手牌の中に整理することがマナーとして求められます。打牌の際に「小手返し」の手法を用いてこれらをうやむやに紛れ込ませる行為は、マナー違反とされています。

  <figcaption class="image-caption">全自動卓には各プレイヤーのリーチ棒置き場があります。画像出典：テレビアニメ『咲 -Saki-』第25話</figcaption>
</figure>

**どのプレイヤーもリーチをかけることができ、すべての供託はその局のアガった人のものとなります。流局した場合、供託はそのまま残され、次に誰かがアガるまで引き継がれてそのアガった人がそれまで蓄積された供託をすべて獲得します。終局まで流局が続いた場合、供託はトップ者のものとなります。**

リーチをしてアガるとそれだけで一役になり、これは日本麻雀において最も出現頻度の高い役です。

<p><YakuName base="立直" ruby="リーチ" />：その局にリーチをかけてアガること。<strong class="colored">1翻</strong>。タンヤオなど他の役と複合する際、しばしば「メン」と略されることがあります（例：リーチ・タンヤオを「メンタン」と略すなど）。</p>

リーチには1翻が付くこと以外に、さらに期待値を高める2つの要素があります。その1つが、偶発役の中で最も出現頻度が高い[^f_yaku]「一発」です。

[^f_yaku]: 出典：天鳳の役統計（半荘戦基準）：<span class="url">https://tenhou.net/sc/prof.html</span>（以下同様）。

<p><YakuName base="一発" ruby="イッパツ" />：リーチをかけた後、1巡以内にアガること。さらに<strong class="colored">1翻</strong>が加算される。</p>

つまり、リーチをかけた後、自分の次のツモ番が回ってくるまでにロンアガリするか、あるいは次のツモ番でツモアガリすることを指します。注意が必要なのは、ここでの「1巡以内」とは、その間に誰のチー・ポン・カン（暗槓を含む）もないことが条件となります。このルールは後述するダブルリーチや地和でも同様です[^keshi]。

[^keshi]: 他家のリーチ後、リスクを顧みずに意図的に鳴きを入れて一発の権利を消滅させる行為を「一発消し」と呼びます。

もう1つのメリットは、リーチをかけたプレイヤーだけが<strong><ruby>裏<rt>ウラ</rt></ruby>ドラ</strong>表示牌をめくる権利を得られる点です。前節で説明したドラ表示牌はすべて上層（上段）にあり、これらを区別するために総称して「**<ruby>表<rt>オモテ</rt></ruby>ドラ**」表示牌と呼びます。そして「裏ドラ表示牌」とは、その表ドラ表示牌の下層（下段）にある牌のことで、リーチ者がアガったときにのみ開示され、リーチ者にだけ効果を持ちます。裏ドラ表示牌も場でのカンが増えるのに伴って増加しますが、すでにめくられている上層の表示牌の下層だけが有効になるため、カンの後に新しく増えた裏ドラは「カン裏ドラ」とも呼ばれます。翻数を数える際、通常の表ドラを「ドラ」と呼び、赤ドラと裏ドラはそれぞれ「アカ」「ウラ」と呼び分けるのが広く使われています。

メンゼンで手役があってテンパイしているものの、あえてリーチをかけない状態を「<ruby>黙<rt>ダマ</rt>聴<rt>テン</rt></ruby>」または「ダマ」と呼びます。しかし、リーチが持つ圧倒的な決定力（特に満貫未満の良形テンパイの場合など）を考えると、メンゼンでの「テンパイ<ruby>即<rt>そく</rt></ruby>リー」が基本であり本手となるケースがほとんどです。

また、リーチに関連する偶然役として以下のようなものもあります。

<p><YakuName base="両立直" ruby="ダブルリーチ" />：自分が第一ツモの前に他家の鳴きが入っていない状況で、1巡目にリーチをかけること。<strong class="colored">2翻</strong>。リーチとは複合しません。一般的には「ダブル立直」や「W立直」と表記され、「ダブリー」と略されます。</p>

> 豆知識：歴史的に「立直」の本来の意味は、1巡目にテンパイを宣言することであり、現在のダブルリーチに近いものでした。

## 1.3.3 フリテン

<ruby>振<rt>フリ</rt>聴<rt>テン</rt></ruby>とは、「<strong class="colored">ロンアガリが禁止される</strong>」テンパイ状態のことです。フリテンになる条件には以下の3つがあります。

1. 捨て牌フリテン：最も基本的なフリテンです。自分の捨て牌（他家に鳴かれた牌も含みます）の中に、自分の待ち牌が1枚でも含まれている場合、その条件が解消されない限りフリテン状態となります。**そのため、テンパイした際は毎回、自分がフリテンになっていないか、ロンアガリができる状態かを確認し、チョンボを防ぐのが賢明です。**
2. リーチ後フリテン：リーチをかけた後にアガリ牌が出たにもかかわらず、これを見逃すと、その局が終了するまでフリテン状態が続きます。
3. 同巡フリテン：他家からアガリ牌が出たにもかかわらず、ロンアガリしなかった場合、次に自分が打牌を完了するまでフリテン状態となります。

フリテンを引き起こしやすい代表的なケースは以下の通りです。

- 過去に<m-pai>1m4m5m</m-pai>から<m-pai>1m</m-pai>を外しており、その後連続して<m-pai>3m</m-pai><m-pai>6m</m-pai><m-pai>2m</m-pai>をツモった結果、最終的に<m-pai>1m4m7m</m-pai>の3面チャンになった場合。すでに<m-pai>1m</m-pai>を切っているため、フリテンとなります。多面チャンはフリテンを誘発しやすいため、特に注意が必要です。ただし、待ちが非常に広い場合は、ツモアガリだけを狙ってあえてフリテンリーチをかけることも多々あります。
- 多面チャンの片方の待ちであれば、より多くの手役や役満が成立するため、高目ではない方の牌が出てもわざと見逃すケース。
- 副露してタンヤオのみの手を進めているものの、最終的に<m-pai>2p3p</m-pai>の形で<m-pai>1p4p</m-pai>を待ってしまった場合。一翻縛りのルールがあるため、<m-pai>1p</m-pai>をツモるか他家から出されても役なしでアガることができず、結果としてフリテンになります。**フリテンは形だけで判定され、役の有無は関係ありません。** このように片方の待ちでしかアガれない状態を「片アガリ」と呼びます。

放銃による失点は非常に大きいため、日本麻雀では守備が極めて重視されます。他家がリーチをかけてきたり、明らかにテンパイしている気配があるときに、自分がまだノーテンであったり、リャンシャンテン以上、あるいは自分の打点が相手より遥かに低い場合は、手牌を崩してでも放銃を避ける「ベタオリ」の姿勢を取るのが一般的です。そして、この守備において最も利用される仕組みこそがフリテンです。最も基本的な守備戦略は、リーチをかけている人の「**<ruby>現<rt>げん</rt>物<rt>ぶつ</rt></ruby>**」、つまり、その人がすでに切っている牌を打つことです。現物は絶対的な<ruby>安<rt>アン</rt>牌<rt>パイ</rt></ruby>です。捨て牌フリテンのルールがあるため、あるプレイヤーの現物を打っても、そのプレイヤーからロンアガリされることは絶対にありません（この状態を牌が「通る」と言います）。同巡フリテンやリーチ後フリテンの仕組みも同様に応用できます。例えば、「同巡に上家が切って通った牌と同じ牌を合わせ打てば、誰に対しても絶対に放銃しない」ということや、「誰かがリーチをかけた後、他家が切って通った牌は、すべてその人に対して通る」ということを推論できます。
