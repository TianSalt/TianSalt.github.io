# 1.3 立直

終於來到了「點題」的一節。在我們這「立直麻雀」中，立直不僅是特色機制，更是比起其他麻雀來說的核心機制。先前說過，立直是狀態役，衹要門前聽牌即可使用，這不僅讓本無手役的牌有了一條默默以門前前進的道路，還可與已有手役的牌疊加。此外，日本麻雀有不少「**門前役**」，衹要手牌有副露便不算做出，使得這些役更多地會與立直一同出現。本節將介紹立直機制以及更多的役，但為了先讓讀者對點數的多寡有個更具體的認識，我們先講講滿貫以上的點數計算。這部分計算最為簡單，也是新手往往最先熟悉的部分⸺本節衹聚焦一般規則，與競技規則的不同之處留待下節說明。

## 1.3.1 滿貫

在實踐中，我們會置1.2.1所講的「基本點」於不顧，衹看點數速查表。不過從原理上說，基本點達到2,000稱作「滿貫」。古典麻雀的滿貫即是封頂最高得點，後來「數值膨脹」，現代的機制是：基本點大於2,000便不再看符數、衹管番數⸺不超過5番，基本點便封頂2,000，而6–7番的手牌基本點固定為3,000，8–10番為4,000，11–12番為6,000，等等。在下一節我們會知道，滿貫以下的「番」確實是字面意思上的兩倍，但顯然滿貫以上增長便急劇放緩。具體地，滿貫以上的點數速查如下表：

> **用法**：以**滿貫**一欄為例，**8,000**：閑家榮和時放銃者應付點數；**2,000-4,000**：閑家自摸和時各閑家、莊家應付點數；**12,000**：莊家榮和時放銃者應付點數；**4,000**：莊家自摸和時各閑家應付點數。オール<sup><i>ōru</i></sup>即英語all。

**役滿**，競技規則文件中又稱四倍滿，可以說是現代日本麻雀的得點上限。完成大三元、小四喜、國士無雙等稀有役時即直接計役滿，此時非役滿役會通通忽略不計，直接取8,000基本點，我們會在下一節詳述；而通過一般的役累計至13番以上叫作「**数え役満**」。關於滿貫和役滿，不同規則又有微妙區別，我們也會在下一節說明。

速查表中得點的表示方法同時也是和牌者報得點所用的說法。如莊家滿貫[^oyaman]榮和報「12,000」，莊家滿貫自摸報「4,000オール」，閑家跳滿自摸報「3,000-6,000」等。如有本場棒，則後接「は<sup><i>wa</i></sup>」再報加計本場棒的點數，如一本場就是「12,000は12,300」「4,000オールは4,100オール」「3,000-6,000は3,100-6,100」等。

## 1.3.2 立直

立直（<l-ja><ruby>立直<rt>リーチ</rt></ruby></l-ja>，雖非詞源但也作reach）是日本麻雀最具特色的機制。門前聽牌即可立直，具體地，任意一次摸牌後，若同時滿足：

1. 門前清（メンゼン）。
2. 本次出牌後可進入聽牌。
3. 持點不少於1,000<small>（若規則有擊飛）</small>。
4. 牌山剩餘可摸的牌不少於4枚<small>（競技規則不設此限，即允許「ツモ番のないリーチ」，衹是摸海底牌後不能立直）</small>。

你可以：

::: theorem
1. 喊「**リーチ<sup><i>rīchi</i></sup>**」。
2. 打出能讓你聽牌的一張牌，橫置在牌河中，叫作「立直宣言牌」<small>（此時若被鳴牌，則下一巡也橫置出牌；若被榮和，不進行第3步）</small>。
3. 交出一根1,000點棒作為<ruby>供<rt>きょう</rt>託<rt>たく</rt></ruby>，置于桌中央，叫作「立直棒」。
4. 直到本局結束，不能挑選要打的牌，衹能摸哪張打哪張（ツモ<ruby>切<rt>ぎ</rt></ruby>り[^tumogiri]）。
:::

[^tumogiri]: 相對地，打出除剛摸到的牌以外的其他手牌叫作「手出し」。

立直供託將歸本局和牌者所有；若流局，供託會一直積攢在桌角的專門區域直到和牌者出現；若直到終局一直流局，則供託歸top者。

立直後和牌即是一役，也是日本麻雀中最高頻的役：

<p><YakuName base="立直" ruby="リーチ" />：該局立直後和牌。計<strong class="colored">1番</strong>。與斷幺等複合時常簡稱「門」（メン），如立直斷幺簡稱「門斷」（メンタン）。</p>

立直除了計1番，還有兩項可能的收益。其一是，有偶然役之中最為高頻[^f_yaku]的「一發」：

[^f_yaku]: 來源為天鳳役統計：<span class="url">https://tenhou.net/sc/prof.html</span>，下同。

<p><YakuName base="一発" ruby="イッパツ" />：立直後一巡內和牌。加計<strong class="colored">1番</strong>。即立直後自己下一次摸牌前榮和，或下一次摸牌時自摸和。必須注意的是，這裏的「一巡」要求期間無人做過任何喫、碰、槓（含暗槓）動作才算，之後要講的兩立直、地和同理。</p>

其二是，衹有立直者才有翻<strong><ruby>裏<rt>ウラ</rt></ruby>ドラ</strong>表示牌的權利。上節講的ドラ表示牌都在上層，為作區分，統稱「**<ruby>表<rt>オモテ</rt></ruby>ドラ**」表示牌，而裏ドラ表示牌即是相應表ドラ表示牌下層的牌，在立直者和牌後揭示，僅對立直者有效。裏ドラ表示牌同樣隨着場上槓的增加而增加，即衹有已翻開的上層表示牌的下層才會被揭示。清點番數時可以將表ドラ叫作「ドラ」，而赤ドラ和裏ドラ分別叫「<ruby>赤<rt>アカ</rt></ruby>」和「<ruby>裏<rt>ウラ</rt></ruby>」。

門前有手役聽牌，但選擇不立直，叫作「<ruby>黙<rt>ダマ</rt>聴<rt>テン</rt></ruby>」「ダマ<sup><i>dama</i></sup>」。不過由於立直強大的威力，尤其滿貫以下的好形，門前「<ruby>聴<rt>テン</rt>牌<rt>パイ</rt>即<rt>そく</rt>立<rt>リー</rt></ruby>」往往是本手。

立直後允許暗槓，但有嚴格的限制，細細究來甚為複雜。

<p><YakuName base="両立直" ruby="ダブルリーチ" />：自己摸牌前無人鳴牌的情況下第一巡立直。計<strong class="colored">2番</strong>。不計立直。常寫作「<l-ja>ダブル立直</l-ja>」，即「double立直」，俗作「<l-ja>W立直</l-ja>」。簡稱「ダブ立」。</p>

## 1.3.4 更多的役

在本小節中我們會介紹完剩餘的所有非役滿役，下一節我們會給出役一覽表。除了役牌和斷幺，也試着注意到其他手役做成機會較高的牌吧。

### 1.3.4.1 一番役

<p><YakuName base="搶槓" ruby="チャンカン" />：他家加槓時，自己恰好聽那張牌，可以榮和。計<strong class="colored">1番</strong>。日語中也寫作「槍槓」。</p>

最稀有的偶然役。搶槓看作一種特殊的榮和，發聲亦是「ロン」，且按榮和結算點數，故意不和也會振聽。若發生搶槓，此槓視為未做成，不揭示新ドラ，也不會影響一發。

偶然役（役滿除外）至此已全部講完：嶺上、海底、河底、一發、ダブ立、搶槓。

<p><YakuName base="門前清自摸和" ruby="メンゼン​ツモ" />：門前狀態下自摸和。計<strong class="colored">1番</strong>。簡稱「自摸」。</p>

最高頻的役之一。除了僅限榮和的搶槓和河底，本役其實能與所有非役滿役複合，謹防漏計。也有「自摸のみ」，即無任何手役但默聽，結果自摸和。門前狀態下，除了自摸和會獎勵1番，榮和也會獎勵10符，見下一節。

<YakuName base="平和" ruby="ピンフ" />：手牌同時滿足

1. 無刻子、槓子，所有面子都是順子。
2. 雀頭不是役牌，即是數牌或客風（オタ<ruby>風<rt>カゼ</rt></ruby>，非場風非自風的風牌）。
3. 聽牌形是五種之中的「兩面」。

<strong class="colored">門前役</strong>，計<strong class="colored">1番</strong>。與符的計算有關，下一節將講述其成因。與斷幺合稱「斷平」，與立直合稱「門平」，與立直、斷幺合稱「門斷平」。門斷平一發自摸就是典型的滿貫牌。

<p><YakuName base="一盃口" ruby="イー​ペーコー" />：擁有兩個完全相同的順子。<strong class="colored">門前役</strong>，計<strong class="colored">1番</strong>。</p>

新手容易忘記一盃口是門前役，實際上由於一盃口屬於順子役，「1番、門前限定」本質上也是一種「有副露則降1番​」（見後文）。事實上立直、自摸、七對等等也都是門前役，但無需特別留意，故本教程介紹時不提及。其實需要留意門前限定的役，衹有平和、一盃口、二盃口、九蓮宝燈四個。

### 1.3.4.5 刻子系

所有刻子役都沒有食い下がり。刻子都可換成槓子。

<p><YakuName base="対々和" ruby="トイトイ​ホー" />：所有面子都是刻子。計<strong class="colored">2番</strong>。簡稱「對對」。</p>
