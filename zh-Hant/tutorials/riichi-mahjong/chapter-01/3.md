# 1.3 立直

終於來到了「點題」的一節。在我們這「立直麻雀」中，立直不僅是特色機制，更是比起其他麻雀來說的核心機制。先前說過，立直是狀態役，衹要門前聽牌即可使用，這不僅讓本無手役的牌有了一條默默以門前前進的道路，還可與已有手役的牌疊加。此外，日本麻雀有不少「門前限定」的役，衹要手牌有副露便不算做出，使得這些役更多地會與立直一同出現。本節將介紹立直機制以及更多的役，但為了先讓讀者對點數的多寡有個更具體的認識，我們先講講滿貫以上的得點計算。這部分計算最為簡單，也是新手往往最先熟悉的部分。

## 1.3.1 滿貫

在實踐中，我們會置1.2.1所講的「基本點」於不顧，衹看得點速查表。不過從原理上說，基本點達到2,000稱作「滿貫」。古典麻雀的滿貫即是封頂最高得點，後來「數值膨脹」，現代的機制是：基本點大於2,000便不再看符數、衹管番數⸺不超過5番，基本點便封頂2,000，而6–7番的手牌基本點固定為3,000，8–10番為4,000，11–12番為6,000，等等。在下一節我們會知道，滿貫以下的「番」確實是字面意思上的兩倍，但顯然滿貫以上增長便急劇放緩。具體地，一般規則下，滿貫以上的得點速查如下表：

> **用法**：以**滿貫**一欄為例，**8,000**：閑家榮和時放銃者應付點數；**2,000-4,000**：閑家自摸和時閑家、莊家應付點數；**12,000**：莊家榮和時放銃者應付點數；**4,000**：莊家自摸和時閑家應付點數。オール<sup><i>ōru</i></sup>即英語all。

「役滿」可以說是現代日本麻雀的得點上限，完成大三元、小四喜、國士無雙等稀有役時即直接計役滿，基本點8,000，我們下一節會介紹；而通過一般的役累計至13番以上叫作「**数え役満**」。關於滿貫和役滿，不同規則又有微妙區別，我們也會在下一節說明。

速查表中得點的表示方法同時也是和牌者報得點所用的說法。如莊家滿貫[^oyaman]榮和報「12,000」，莊家滿貫自摸報「4,000オール」，閑家跳滿自摸報「3,000-6,000」等。如有本場棒，則後接「は<sup><i>wa</i></sup>」再報加計本場棒的點數，如一本場就是「12,000は12,300」「4,000オールは4,100オール」「3,000-6,000は3,100-6,100」等。

## 1.3.2 立直

立直（<l-ja><ruby>立直<rt>リーチ</rt></ruby></l-ja>，雖非詞源但也作reach）是日本麻雀最具特色的機制。門前聽牌即可立直，具體地，任意一次摸牌後，若同時滿足：

1. 門前清（メンゼン）。
2. 本次出牌後可進入聽牌。
3. 持點不少於1,000<small>（若規則有擊飛）</small>。
4. 牌山剩餘可摸的牌不少於4枚<small>（競技規則不設此限，即允許「ツモ番のないリーチ」，衹是摸海底牌後不能立直）</small>。

你可以：

::: theorem
1. 喊「**リーチ<sup><i>rīchi</i></sup>**」。
2. 打出能讓你聽牌的一張牌，橫置在牌河中，這張牌叫作「立直宣言牌」<small>（若立直宣言牌被他家鳴牌，則下一巡打出的牌橫置）</small>。
3. 若無人榮和立直宣言牌，你交出一根1,000點棒（立直棒）作為<ruby>供<rt>きょう</rt>託<rt>たく</rt></ruby>，置于桌中央（若有專門的凹槽則置於其中）。
4. 直到本局結束，不能挑選要打的牌，衹能摸哪張打哪張（ツモ<ruby>切<rt>ぎ</rt></ruby>り[^tumogiri]）。
:::

[^tumogiri]: 相對地，打出除剛摸到的牌以外的其他手牌叫作「手出し」。

立直供託將歸本局和牌者所有；若流局，供託會一直積攢在桌角的專門區域直到和牌者出現；若直到終局一直流局，則供託歸top者。

立直後和牌即是一役：

<YakuName base="立直" ruby="リーチ" />：立直後和牌。計<strong class="colored">1番</strong>。與其他役複合時常簡稱「門」（メン），如立直斷幺簡稱「門斷」（メンタン）。

立直還能獲得其他收益，其一是與立直相關的高頻[^f_yaku]偶然役「一發」：

[^f_yaku]: 天鳳役統計：<span class="url">https://tenhou.net/sc/prof.html</span>

<YakuName base="一発" ruby="イッパツ" />：立直後一巡內和牌。加計<strong class="colored">1番</strong>。即立直後下一次摸牌前榮和或下一次摸牌時自摸和。要求期間無人做過任何鳴牌（含暗槓）動作才算，**之後要講的兩立直、地和同理**。

其二是衹有立直者才有翻<strong><ruby>裏<rt>うら</rt></ruby>ドラ</strong>表示牌的權利。上節講的ドラ表示牌都在上層，為作區分，統稱「**<ruby>表<rt>おもて</rt></ruby>ドラ**」表示牌，而裏ドラ表示牌即是相應表ドラ表示牌下層的牌，當立直者和牌後即揭示，僅對立直者有效。裏ドラ表示牌同樣隨着場上槓的增加而增加，即衹揭示所有已翻開的上層表示牌的下層。清點番數時可以將表ドラ叫作「ドラ」，而赤ドラ和裏ドラ分別叫「アカ<sup><i>aka</i></sup>」和「ウラ<sup><i>ura</i></sup>」。

立直後允許暗槓，但有嚴格的限制。

## 1.3.4 更多的役

### 1.3.4.1 剩餘的1番役

<p><YakuName base="門前清自摸和" ruby="メンゼン​ツモ" />：門前狀態下自摸和。計<strong class="colored">1番</strong>。簡稱「自摸」。</p>

常與立直複合，立直自摸必多計此役，若是<ruby>黙<rt>ダマ</rt>聴</ruby>（門前聽牌但不立直）自摸也計此役。本是無役的默聽，若自摸和，因有此役，也可和牌。門前狀態下榮和也有獎勵，但不是1番，而是10符，見下一節。

<YakuName base="平和" ruby="ピンフ" />：<strong class="colored">門前限定</strong>。手牌滿足

1. 無刻子、槓子；
2. 雀頭是數牌或客風（オタ<ruby>風<rt>カゼ</rt></ruby>，非場風非自風的風牌），即雀頭非役牌；
3. 聽牌形是五種之中的兩面形。

計<strong class="colored">1番</strong>。與符的計算有關，將在下一節詳述，此處暫作羅列。

<YakuName base="一盃口" ruby="イー​ペーコー" />：<strong class="colored">門前限定</strong>。擁有兩個相同的順子。計<strong class="colored">1番</strong>。簡稱「一盃」。

::: details 2. 得點計算<br>東二局1本場　東家　立直<br>ドラ表示牌<m-pai>0s</m-pai>　裏ドラ表示牌<m-pai>4z</m-pai><p><m-tehai>3m4m4m0m5m4s5s6s6s6s6p7p8p　3m</m-tehai>榮和</p>
7番（立直・平和・一盃口・ドラ3・赤）

跳滿　6,000オールは6,100オール
:::

新手容易忘記一盃口是門前限定役，實際上由於一盃口屬於順子系役，「1番、門前限定」本質上也是一種「食い下がり​」（見後文）。

<YakuName base="搶槓" ruby="チャンカン" />：榮和他家的加槓牌。計<strong class="colored">1番</strong>。

最稀有的偶然役。搶槓屬於一種特殊的榮和，發聲亦是「ロン」，且按榮和結算點數。放銃家的槓視作未做成，不揭示新ドラ。
