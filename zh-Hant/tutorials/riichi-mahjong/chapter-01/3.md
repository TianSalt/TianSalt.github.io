# 1.3 立直

終於來到了「點題」的一節。在我們這「立直麻雀」中，立直不僅是特色機制，更是比起其他麻雀來說的核心機制。先前說過，立直是狀態役，衹要門前聽牌即可使用，這不僅讓本無手役的牌有了一條默默以門前前進的道路，還可與已有手役的牌疊加。此外，日本麻雀有不少「**門前役**」，衹要手牌有副露便不算做出，使得這些役更多地會與立直一同出現。本節將介紹立直機制以及更多的役，但甜Salt首先想再講講點數相關的話題，畢竟日本麻雀比拼的是終局順位，而順位由持點決定。為了對一副手牌的得點多寡⸺在戰術上常叫「<ruby>打<rt>だ</rt>点<rt>てん</rt></ruby>」⸺有個具體的認識，我們先講講滿貫以上的得點計算。這部分計算最為簡單，也是新手往往最先熟悉的部分⸺本節衹聚焦一般規則，與競技規則的不同之處留待下節說明。

## 1.3.1 滿貫

在實踐中，我們會置1.2.1所講的「基本點」於不顧，衹看點數速查表。不過從原理上說，基本點達到2,000稱作「滿貫」。古典麻雀的滿貫即是封頂最高得點，後來「數值膨脹」，現代的機制是：基本點大於2,000便不再看符數、衹管番數⸺不超過5番，基本點便封頂2,000，而6–7番的手牌基本點固定為3,000，8–10番為4,000，11–12番為6,000，等等。在下一節我們會知道，滿貫以下的「番」確實是字面意思上的兩倍，但顯然滿貫以上增長便急劇放緩。具體地，滿貫以上的點數速查如下表：

## 1.3.2 立直

立直（<l-ja><ruby>立直<rt>リーチ</rt></ruby></l-ja>，雖非詞源但也作reach）是日本麻雀最具特色的機制。門前聽牌即可立直（リーチする・リーチをかける），具體地，任意一次摸牌後，若同時滿足：

1. 門前（沒有喫、碰、明槓）。
2. 已可聽牌。
3. 持點不少於1,000<small>（若規則有擊飛）</small>。
4. 牌山剩餘可摸的牌不少於4枚<small>（競技規則不設此限，即允許「ツモ番のないリーチ」，衹是摸海底牌後不能立直）</small>。

你可以：

::: theorem
1. 喊「**リーチ<sup><i>rīchi</i></sup>**」。
2. 打出能讓你聽牌的一張牌，橫置在牌河中，叫作「立直宣言牌」<small>（此時若被鳴牌，則下一巡也橫置出牌；若被榮和，不進行第3步）</small>。
3. 支付一根1,000點棒作為<ruby>供<rt>きょう</rt>託<rt>たく</rt></ruby>，置于桌中央，叫作「立直棒」。
4. 直到本局結束，不能挑選要打的牌，衹能摸哪張打哪張（ツモ<ruby>切<rt>ぎ</rt></ruby>り[^tumogiri]）。
:::

[^tumogiri]: 相對地，打出除剛摸到的牌以外的其他手牌叫作「手出し」（漢語玩家圈叫「手切」）。為區分二者，日本麻雀一般要求先出牌，再把剛摸到的牌整理進手牌中。使用「<l-ja>小手返し</l-ja>」的手法悄咪咪混淆二者是有違禮儀的行為。

**每家都可立直，所有供託歸本局和牌者所有；若流局，供託會一直積攢，直到和牌者出現將本局及之前積攢的供託全部取走；若一直流局到終局，供託歸top者。**

立直後和牌即是一役，也是日本麻雀中最高頻的役：

<p><YakuName base="立直" ruby="リーチ" />：該局立直後和牌。計<strong class="colored">1番</strong>。與斷幺等複合時常簡稱「<ruby>門<rt>メン</rt></ruby>」，如立直斷幺簡稱「<ruby>門<rt>メン</rt>断<rt>タン</rt></ruby>」。</p>

立直除了計1番，還有兩項可能的收益。其一是，有偶然役之中最為高頻[^f_yaku]的「一發」：

[^f_yaku]: 來源為天鳳役統計，以半莊戰為準：<span class="url">https://tenhou.net/sc/prof.html</span>，下同。

<p><YakuName base="一発" ruby="イッパツ" />：立直後一巡內和牌。加計<strong class="colored">1番</strong>。</p>

即立直後自己下一次摸牌前榮和，或下一次摸牌時自摸和。必須注意的是，這裏的「一巡」要求期間無人做過任何喫、碰、槓（含暗槓）動作才算，之後要講的兩立直、地和同理[^keshi]。

[^keshi]: 他家立直後不顧風險故意鳴牌以消除一發叫做「一発消し」（漢語玩家圈叫「破一發」）。

其二是，衹有立直者才有翻<strong><ruby>裏<rt>ウラ</rt></ruby>ドラ</strong>表示牌的權利。上節講的ドラ表示牌都在上層，為作區分，統稱「**<ruby>表<rt>オモテ</rt></ruby>ドラ**」表示牌，而裏ドラ表示牌即是相應表ドラ表示牌下層的牌，在立直者和牌後揭示，僅對立直者有效。裏ドラ表示牌同樣隨着場上槓的增加而增加，即衹有已翻開的上層表示牌的下層才會生效，隨槓而新增的裏ドラ也被叫作「槓裏ドラ」。清點番數時可以將表ドラ叫作「ドラ」，而赤ドラ和裏ドラ分別叫「<ruby>赤<rt>アカ</rt></ruby>」和「<ruby>裏<rt>ウラ</rt></ruby>」。

門前有手役聽牌，但選擇不立直，叫作「<ruby>黙<rt>ダマ</rt>聴<rt>テン</rt></ruby>」「ダマ<sup><i>dama</i></sup>」。不過由於立直強大的威力，尤其滿貫以下的好形，門前「<ruby>聴<rt>テン</rt>牌<rt>パイ</rt>即<rt>そく</rt>立<rt>リー</rt></ruby>」常是本手。

還有一個與立直相關的偶然役：

<p><YakuName base="両立直" ruby="ダブルリーチ" />：自己摸牌前無人鳴牌的情況下第一巡立直。計<strong class="colored">2番</strong>。不計立直。常寫作「<l-ja>ダブル立直</l-ja>」，即「double立直」，俗作「<l-ja>W立直</l-ja>」。簡稱「ダブリー」。</p>

> 冷知識：歷史上「立直」的本義是第一巡就告聽牌，有點類似現在的<l-ja>ダブル立直</l-ja>。

## 1.3.3 振聽

振聽（フリテン）是「<strong class="colored">禁止榮和</strong>」的聽牌。有三種觸發條件：

1. 捨牌振聽（捨て牌フリテン）：最默認的含義。自己的捨牌（包括被鳴了的）中有自己所聽的牌之一或更多，則進入振聽狀態，直到不滿足條件。**因此每次聽牌後，最好先檢查自己有無振聽、能否榮和，以免犯規。**
2. 立直振聽（リーチ後フリテン）：立直後能和而不和（**<l-ja>見逃す</l-ja>**），則進入振聽狀態直到本局結束。
3. 同巡振聽（<l-ja>同巡フリテン</l-ja>）：能榮和而不榮和，則進入振聽狀態，直到自己下一次出牌。

一些常見的造成振聽的情景：

- 曾拆掉<m-pai>1m4m5m</m-pai>中的<m-pai>1m</m-pai>，後來又接連摸到了<m-pai>3m</m-pai><m-pai>6m</m-pai><m-pai>2m</m-pai>，最後變成了聽<m-pai>1m4m7m</m-pai>三面，因打過<m-pai>1m</m-pai>而振聽。多面張很容易觸發振聽，應特別注意。不過如果聽牌面很廣，很多時候會直接選擇振聽立直，衹求自摸。
- 多面聽的某一邊能達成更多手役甚至役滿，從而來另一邊的牌故意不和。
- 副露做斷幺のみ，但最後用<m-pai>2p3p</m-pai>聽<m-pai>1p4p</m-pai>，囿於一番起和，摸到或者別人打出<m-pai>1p</m-pai>會無役不能和，這樣也會振聽。<strong>振聽衹論形，不論役。</strong>這種衹和一邊<sub>兒</sub>的牌叫作「片アガリ」（漢語玩家圈叫「偏聽」）。

因為放銃代價高昂，日本麻雀十分注重防守，他家立直家或者明顯已聽牌時，如果自己未聽牌甚至兩向聽以上，或者打點遠低於對方，往往會選擇完全棄和（ベタオリ）的姿態，把手牌拆散也要防止放銃。而振聽恰恰是防守最常利用的機制，最基本的防守策略就是打立直家的「**<ruby>現<rt>げん</rt>物<rt>ぶつ</rt></ruby>**」⸺他出過的牌。現物是絕對的<ruby>安<rt>アン</rt>牌<rt>パイ</rt></ruby>，由於捨牌振聽規則的存在，打某家的現物完全不可能被該家榮和［我們說這張牌能「通過」（<l-ja>通る</l-ja>）］。後兩種振聽的運用方式類似，我們可以推論出，跟打同巡上家打的那張牌一定不會放銃任何人，而一家立直後其他家打過的所有牌，都已能通過該立直家。

## 1.3.4 更多的役

在本小節中我們會介紹完剩餘的所有非役滿役，下一節我們會給出役一覽表。除了役牌和斷幺，也試着注意到其他手役做成機會較高的場景吧。

### 1.3.4.1 一番役

<p><YakuName base="搶槓" ruby="チャンカン" />：他家加槓時，自己恰好聽那張牌，可以榮和。計<strong class="colored">1番</strong>。日語中也寫作「槍槓」。</p>

最稀有的偶然役。搶槓看作一種特殊的榮和，發聲亦是「ロン」，且按榮和結算點數，故意不和也會振聽。若發生搶槓，此槓視為未做成，不揭示新ドラ，也不會影響一發。

偶然役（役滿除外）至此已全部講完：嶺上、海底、河底、一發、W立、搶槓。

<p><YakuName base="門前清自摸和" ruby="メンゼンチンツモホー" />：門前狀態下自摸和。計<strong class="colored">1番</strong>。簡稱「自摸」。</p>

最高頻的役之一。除了僅限榮和的搶槓和河底，本役其實能與所有非役滿役複合，謹防漏計。也有「自摸のみ」，即無任何手役但默聽，結果自摸和。門前狀態下，除了自摸和會獎勵1番，榮和也會獎勵10符，見下一節。

<YakuName base="平和" ruby="ピンフ" />：手牌同時滿足

1. 無刻子、槓子，即四個面子都是數牌順子。
2. 雀頭不是役牌，即是數牌或客風（オタ<ruby>風<rt>カゼ</rt></ruby>，非場風非自風的風牌）。
3. 聽牌形是五種之中的「兩面」。

<strong class="colored">門前役</strong>，計<strong class="colored">1番</strong>。與斷幺合稱「断平」，與立直合稱「門平」，與立直、斷幺合稱「門断平」。門断平一発自摸就是典型的滿貫牌。
