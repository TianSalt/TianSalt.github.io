# 1.1 基本機制

立直麻雀是運氣成分極大、但通過數百場以上的對局數據又能高下立判的遊戲。本教程基於21世紀初、特別是網絡麻雀（<l-ja>ネット麻雀</l-ja>）興起後更加定型的日本麻雀一般規則。而以M.LEAGUE（Ｍリーグ）為代表的競技麻雀規則與一般規則有諸多細節上的不同，也會在講解中提及。

日本麻雀的用語同遊戲本身一道由中國傳來，相關漢字詞讀音很像是來自北京官話。此外，不少用語有多個同義叫法，我雖然會儘量提及，也難免掛一漏萬，讀者在語料中遇到時莫要驚訝。

## 1.1.2 遊戲的進行

我們先從<ruby>配<rt>ハイ</rt>牌<rt>パイ</rt></ruby>已完成之後開始說起。配牌完成後，每位玩家會有13張<ruby>手<rt>て</rt>牌<rt>はい</rt></ruby>，比如：

<m-tehai>1m1m6m9m3p4p4p0p9p2s4s1z3z</m-tehai>

隨後每位玩家會輪流<ruby>摸<rt>モウ</rt>打<rt>ダ</rt></ruby>：

- <strong>摸牌</strong>（<ruby>自摸<rt>ツモ</rt></ruby>[^tsumo]）：首先，摸進現存的<ruby>牌<rt>ハイ</rt>山<rt>ヤマ</rt></ruby>[^yama]最前面的一張牌；
- <strong><ruby>打<rt>ダ</rt>牌<rt>ハイ</rt></ruby></strong>：然後，選擇一張要捨棄的手牌打出（<l-ja>切る・捨てる</l-ja>）。

[^yama]: 學名<sub>兒</sub><ruby>壁<rt>ピー</rt>牌<rt>パイ</rt></ruby>，俗稱<ruby>山<rt>ヤマ</rt></ruby>，漢語一般就叫「牌山」。

[^tsumo]: 「自摸」在漢語中衹指「自摸和」（後述）而無摸牌的意思，日語中則都稱為「ツモ」，摸牌還可用動詞「ツモる」。

麻雀是四人遊戲（雖然在日本也流行三人麻雀，暫且不提），四人圍坐在方桌旁，**逆時針**輪流摸打。除自己之外的其他三家稱為「<ruby>他<rt>ター</rt>家<rt>チャ</rt></ruby>」。自己左邊叫作「<ruby>上<rt>カミ</rt>家<rt>チャ</rt></ruby>」，上家摸打完就輪到自己，自己摸打完輪到右面的「<ruby>下<rt>シモ</rt>家<rt>チャ</rt></ruby>」，而對面則叫作「<ruby>対<rt>トイ</rt>面<rt>メン</rt></ruby>」（漢語叫「對家」）。四人摸打一圈稱為「<ruby>一巡<rt>いちじゅん</rt></ruby>」。

每一局有一人是「莊家」，日語通稱「<ruby>親<rt>おや</rt></ruby>」，其餘人則稱為「閑家」或「散家」，日語通稱「<ruby>子<rt>こ</rt></ruby>」。絕對方位上，莊家叫作「<ruby>東<rt>トン</rt>家<rt>チャ</rt></ruby>」，其下家叫作「<ruby>南<rt>ナン</rt>家<rt>チャ</rt></ruby>」，再下家叫作「<ruby>西<rt>シャー</rt>家<rt>チャ</rt></ruby>」，再下家叫作「<ruby>北<rt>ペー</rt>家<rt>チャ</rt></ruby>」（所以麻雀是「上北下南左東右西」的世界）。

若有任一方達成<ruby>和<rt style="font-size: 0.7em;">hú</rt></ruby>牌[^hule]，一局即告結束，結算分數（<ruby>点<rt>てん</rt>数<rt>すう</rt></ruby>），然後將所有牌重新洗牌開始下一局。「局」相關規則之後還會詳述，我們先來看如何達成和牌。

[^hule]: 漢語中，<ruby>和<rt>hú</rt></ruby>牌的玩家會喊「<ruby>和<rt>hú</rt>了<rt>le</rt></ruby>」，而日語直接稱和牌為「<ruby>和<rt>ホー</rt>了<rt>ラ</rt></ruby>」或「<ruby>和了<rt>あがり</rt></ruby>」（另有「<ruby>和了<rt>あが</rt>り</ruby>」等寫法），和了時喊的則是「ロン」與「ツモ」，後述。

## 1.1.3 和牌與聽牌

### 1.1.3.1 和牌形

三張相同的牌，稱為「**<ruby>刻子<rt>コーツ</rt></ruby>**」，如：

<m-tehai>3p3p3p</m-tehai>、<m-tehai>6z6z6z</m-tehai>……

三張同種、相鄰的數牌，稱為「**<ruby>順子<rt>シュンツ</rt></ruby>**」，如：

<m-tehai>3p4p5p</m-tehai>、<m-tehai>6s7s8s</m-tehai>……

兩張相同的牌，稱為「**對子**」（<ruby>対子<rt>トイツ</rt></ruby>）：

<m-tehai>9m9m</m-tehai>、<m-tehai>1z1z</m-tehai>……

「刻子」與「順子」統稱「**<ruby>面子<rt>メンツ</rt></ruby>**」。麻雀的目標是做成「**4個面子 + 1個對子**」共14張牌組成的「<ruby>和<rt>hú</rt></ruby>牌形」（<ruby>和<rt>ホー</rt>了<rt>ラ</rt>形<rt>ケイ</rt></ruby>），如：

<m-tehai>4m5m6m4p4p4p6p7p8p2s3s4s8s8s</m-tehai>

我們常稱為「**四面子一<ruby>雀頭<rt>ジャントウ</rt></ruby>**」。「雀頭」即對子，是專指和牌形中的對子，俗稱「アタマ」（頭），漢語如今多稱為「將」（廣東稱「眼」）。

### 1.1.3.2 聽牌

正如我們之前所說，配牌完成後，每位玩家會有13張牌，每一巡都會摸進第14張，再捨棄一張變回13張。最終，在某次摸牌後達成14張的和了形，宣告和牌。如果自己的13張手牌已是「下次摸牌時就可能和牌」的狀態，稱為「**聽牌**」（<ruby>聴<rt>テン</rt>牌<rt>パイ</rt></ruby>）：

<m-tehai>4m5m4p4p4p6p7p8p2s3s4s8s8s</m-tehai>

所聽之牌，日語稱為「<ruby>和了<rt>アガリ</rt>牌<rt>ハイ</rt></ruby>」或「待ち牌」，比如這手牌是<m-pai>3m</m-pai>或<m-pai>6m</m-pai>（聽「<ruby>三<rt>サブ</rt>六<rt>ロー</rt></ruby>萬」），因為來<m-pai>3m</m-pai>會組成<m-pai>3m4m5m</m-pai>的順子，而來<m-pai>6m</m-pai>會組成<m-pai>4m5m6m</m-pai>的順子，是「**兩面聽**」（<ruby>両<rt>リャン</rt>面<rt>メン</rt></ruby>待ち・両面<ruby>張<rt>チャン</rt></ruby>）。當然，也有三面聽甚至更多面聽，比如：

<m-tehai>3m4m5m6m7m6p7p8p2s3s4s8s8s　</m-tehai><span class="tehaitext">聽</span><m-tehai>2m5m8m</m-tehai>

聽牌形的分類關係着之後要講的「符」的計算，而這手牌其實也是分類到「兩面」形的⸺如果來的是<m-pai>2m</m-pai>，可看作手持<m-pai>3m4m　5m6m7m</m-pai>，聽<m-pai>2m5m</m-pai>；如果來的是<m-pai>8m</m-pai>，可看作手持<m-pai>3m4m5m　6m7m</m-pai>，聽<m-pai>5m8m</m-pai>；而如果來的是<m-pai>5m</m-pai>，看成兩種中的哪一種都可以。

### 1.1.3.3 和牌

一旦聽牌，除了通過自己摸上所聽的牌來和牌，即**自摸和**（ツモホー，通稱ツモアガリ），還可以和別家打出的牌，稱作**榮和**（ロンホー，通稱ロンアガリ）。自摸和時，比如上面這手牌你終於在某一巡摸到了<m-pai>4m</m-pai>或<m-pai>7m</m-pai>，那麼你喊「**ツモ<sup>_tsumo_</sup>**」，並開示自己全部的手牌，結算分數，結束本局；而榮和時，即你看到別家打出了<m-pai>4m</m-pai>或<m-pai>7m</m-pai>，你喊「**ロン<sup>_ron_</sup>**」，當然也要開示自己全部的手牌，結算分數，結束本局。

在日本麻雀中，除了做成和牌形之外，還需要「有役」和「不違反振聽規則」才能真正和牌，但核心仍是和牌形。另外兩項要求，本教程會帶你自然地學會。

## 1.1.4 聽牌形

按照「四面子一雀頭」的和牌形，除了上面這樣用兩張相鄰數牌組成的「兩面聽」之外，還有四種可能的聽牌形：

- 如果手牌已完成三面子一雀頭，剩下了兩張間隔為一的同種數牌，則稱為「坎張聽」或「嵌張聽」（カンチャン[^kan]待ち，廣東稱「卡窿」）：

  <m-tehai>1s2s3s1p3p1m2m3m3m3m9m9m9m　</m-tehai><span class="tehaitext">聽「坎<m-pai>2p</m-pai>」</span>

  有沒有看出這手牌的雀頭是什麼呢？
  可以發現，兩面可以等兩種牌，坎張衹能等一種，不如兩面容易和牌。

- 如果最後剩下的是12或者89，即「兩面，但麻雀裏沒有零和十，所以衹是在等三或者七」，則稱為聽「邊張」（ペンチャン待ち）：

  <m-tehai>8s9s2p3p3p3p4p1z1z1z2z2z2z　</m-tehai><span class="tehaitext">聽「邊<m-pai>7s</m-pai>」</span>

  牌按大小排序了，其實餅子部分是順子<m-pai>2p3p4p</m-pai>及雀頭<m-pai>3p3p</m-pai>。

- 如果手牌已完成三個面子，剩下的是兩個對子，來哪個都可以讓自己成刻子、讓另一個對子當雀頭，則稱為聽「雙碰」（シャンポン待ち・シャボ待ち）：

  <m-tehai>5p0p3s4s0s6m7m7m8m8m9m1z1z　</m-tehai><span class="tehaitext">聽「<m-pai>5p</m-pai>和<m-pai>1z</m-pai>的雙碰<sub>兒</sub>」</span>

  萬子部分其實是兩個順子，你看出來了嗎？

- 如果手牌已完成四個面子，剩下一張孤零零的牌等着另一張來成雀頭，則稱為聽「單騎」（タンキ待ち）：

  <m-tehai>1s3s3s3s4s4s4s6s6s6s8s8s8s　</m-tehai><span class="tehaitext">聽「<m-pai>1s</m-pai>單騎」</span>

  好吧其實我舉了一個不單純是單騎的例子。這手其實是聽<m-pai>1s2s</m-pai>，因為還可以把<m-pai>3s3s</m-pai>看成雀頭，聽坎<m-pai>2s</m-pai>。

[^kan]: 漢字寫法是「嵌張」。

::: details 最後這例如果把<m-pai>1s</m-pai>換成<m-pai>2s</m-pai>：<p><m-tehai>2s3s3s3s4s4s4s6s6s6s8s8s8s</m-tehai></p>會是個很廣的聽牌。你能找出全部可以和的牌嗎？<p>（點擊展開答案）</p>

<m-pai>1s2s3s4s</m-pai>四面聽。

首先把<m-pai>4s4s4s</m-pai>看成一個刻子，此時<m-pai>2s3s3s3s</m-pai>既可以看成雀頭<m-pai>3s3s</m-pai> + <m-pai>2s3s</m-pai>，聽兩面<m-pai>1s4s</m-pai>，也可以看成刻子<m-pai>3s3s3s</m-pai> + <m-pai>2s</m-pai>單騎；

然而<m-pai>2s　3s　4s</m-pai>的部分還可以看作<n-w><m-pai>2s3s4s</m-pai><m-pai>3s3s</m-pai><m-pai>4s4s</m-pai>，</n-w>這是<m-pai>3s4s</m-pai>雙碰。
:::

雖然雙碰確實也是「兩」面，但在討論五類聽牌形（待ちの形式）時是單獨看待的。從牌理上看，雙碰是比兩面更難和牌的形狀⸺麻雀每種牌有且僅有4張，如果拿第一個<m-pai>5m6m</m-pai>聽<m-pai>4m7m</m-pai>的例子，與<m-pai>5p0p1z1z</m-pai>聽<m-pai>5p1z</m-pai>的例子作比較，前者的聽牌面是還剩4張的<m-pai>4m</m-pai>和還剩4張的<m-pai>7m</m-pai>共8張牌，而後者因為所聽的兩種牌已各手持2張，外面就衹剩2張<m-pai>5p</m-pai>和2張<m-pai>1z</m-pai>共4張牌，這與坎張⸺如<m-pai>1p3p</m-pai>聽4枚的<m-pai>2p</m-pai>⸺的枚數是一樣的。總之，單從枚數而言，不考慮<ruby>場況<rt>ばきょう</rt></ruby>（牌河等的狀況），那麼最佳的聽牌形是兩面，其次是雙碰、坎張和邊張，最差的是單騎。我們稱兩面張（及<ruby>多<rt>タ</rt></ruby>面張）為「**<ruby>好<rt>こう</rt>形<rt>けい</rt></ruby>**」，另外幾種為「**<ruby>愚<rt>ぐ</rt>形<rt>けい</rt></ruby>**」。
