# 序

歡迎來到立直麻<ruby>雀<sub>兒</sub><rt>jiàng</rt></ruby>[^1]教程系列。「立直麻雀」（<l-ja>リーチ<ruby>麻雀<rt>マージャン</rt></ruby></l-ja>）現今已大可看作「日本麻雀」的同義詞，這一名號來自其特色規則「立直」。

[^1]: 「麻雀<sub>兒</sub>」即「麻將」。一般認為，「將」即北部吳語「雀」的兒化，參見：https://en.wikipedia.org/wiki/Erhua#Wu

該系列計劃隨我的麻雀學習一同更新，先從基本規則開始，然後介紹策略。本教程同樣適合沒有接觸過任何麻雀的讀者，也將包含邊邊角角的細則，以及一些關於「所以然」的東西，新手可酌情跳過。此外，本教程特別面向想要學會用日語打麻雀和探討麻雀的讀者，非此類讀者可能會感到這些日語相關註解無處不在、阻礙閱讀，敬請諒解。
