# 無障礙聲明

## 我們的承諾

我們致力於確保殘障人士的數字無障礙訪問。我們不斷改善每個人的用戶體驗，並應用相關的無障礙標準。

## 符合性狀態

我們的目標是符合 Web 內容無障礙指南（WCAG）2.1 AA 級標準。這些指南解釋了如何讓殘障人士更容易訪問網頁內容。

## 障礙特性

### 鍵盤導航
- 所有交互元素都可以通過鍵盤訪問
- 提供跳轉到主內容的鏈接
- 整個網站的邏輯 Tab 鍵順序

### 屏幕閱讀器支持
- 語義化 HTML 結構
- 交互元素的 ARIA 標籤
- 圖片的替代文本
- 正確的標題層次結構

### 視覺設計
- 足夠的顏色對比度
- 文本可調整大小而不損失功能
- 清晰一致的導航
- 深色模式支援

### 動畫和運動
- 尊重 `prefers-reduced-motion` 設定
- 無自動播放內容

### 多語言支援
- 為每個頁面正確設定語言屬性
- 鍵盤可訪問的語言切換器

## 輔助技術兼容性

本網站設計兼容：
- 屏幕閱讀器（JAWS、NVDA、VoiceOver）
- 屏幕放大軟體
- 語音識別軟體
- 僅鍵盤導航

## 已知問題

我們正在積極識別和解決任何無障礙障礙。如果您遇到任何問題，請聯繫我們。

## 反饋

我們歡迎您對本網站的無障礙性提供反饋。如果您遇到任何障礙，請告知我們：

- **電子郵件**: tiansalt@outlook.com
- **GitHub**: [創建問題](https://github.com/TianSalt/TianSalt.github.io/issues)

我們嘗試在 2 個工作日內回覆反饋。

## 技術規格

無障礙特性使用以下技術構建：
- 語義化 HTML5 元素
- 必要時使用 ARIA 屬性
- CSS 用於視覺呈現
- 漸進增強原則

## 評估方法

本網站使用以下方法評估：
- 自動化測試工具（axe、Lighthouse）
- 手動鍵盤測試
- 屏幕閱讀器測試
- 顏色對比度分析器
